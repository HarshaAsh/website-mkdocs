    ఉపయోగం కేస్ అవలోకనం¶

    భారతదేశం యొక్క విస్తృత రైల్వే నెట్‌వర్క్‌లో రైలు ఆలస్యాలు సాధారణ సవాలు, ఇది రోజుకు కోట్ల మంది ప్రయాణికులను ప్రభావితం చేస్తుంది. ఈ ఆలస్యాల వెనుక ఉన్న నమూనాలను అర్థం చేసుకోవడం షెడ్యూలింగ్‌ను ఆప్టిమైజ్ చేయడంలో మరియు ఆపరేషనల్ సామర్థ్యాన్ని మెరుగుపరచడంలో సహాయపడుతుంది. ఈ బ్లాగ్‌లో, మేము నిజ సమయ రైలు నడుస్తున్న స్థితి డేటాను సేకరించాము, ఆలస్యాల మేట్రిక్‌లను లెక్కించాము మరియు స్టేషన్లలో ఆలస్యాల భౌగోళిక మరియు కాలిక నమూనాలను గుర్తించడానికి క్లస్టరింగ్ సాంకేతికతలను ఉపయోగించాము. లక్ష్యం ఆలస్యాలు తరచుగా జరుగుతున్న హాట్‌స్పాట్‌లను కనుగొనడం మరియు వాటి తీవ్రతను విశ్లేషించడం.

    from bs4 import BeautifulSoup
import requests
import pandas as pd

    BeautifulSoup తో వెబ్ స్క్రాపింగ్¶

    రైలు ఆలస్యాల డేటాను సేకరించడానికి, మేము HTML మరియు XML డాక్యుమెంట్లను పార్సింగ్ చేయడానికి ఉపయోగించే Python లైబ్రరీ అయిన BeautifulSoup ను ఉపయోగించాము. డేటా runningstatus.in నుండి సేకరించబడింది, ఇది ప్రత్యక్ష రైలు స్థితి నవీకరణలను అందిస్తుంది. BeautifulSoup మాకు అనుమతించింది:
    1. స్టేషన్ పేర్లు, షెడ్యూల్ మరియు వాస్తవ రాక/ప్రయాణ సమయాలు, మరియు ఆలస్య సమాచారం కలిగిన HTML పట్టికల నుండి నిర్మాణాత్మక డేటాను తీసుకోవడం.
    2. HTML ట్యాగ్‌లలో అసమానతలను నిర్వహించడం మరియు స్టేషన్ కోడ్స్ కోసం <abbr> మరియు ఆలస్య స్థితి కోసం <small> వంటి నెస్టెడ్ అంశాలను పార్సింగ్ చేయడం.
    3. సేకరించిన డేటాను pandas DataFrame గా మార్చడం, తదుపరి విశ్లేషణ కోసం.

    ఈ విధానం వేల రైళ్ల కోసం డేటా సేకరణను ఆటోమేటెడ్ చేసింది, 26,000 కంటే ఎక్కువ వరుసలతో సమృద్ధిగా ఉన్న డేటాసెట్‌ను సృష్టించింది.

    Janmabhoomi express కోసం, మాకు ప్రత్యక్ష నడుస్తున్న స్థితిలో క్రింది సమాచారం ఉంది:

    తదనుగుణంగా HTML కోడ్:

    def get_table(train_number):
    # లింక్ రెండు ఇన్‌పుట్‌లను తీసుకుంటుంది, రైలు సంఖ్య మరియు తేదీ YYYYMMDD (2025-12-13)
    link = 'https://runningstatus.in/status/'+ str(train_number)+'-on-20251213'
    try:
        response = requests.get(link, timeout=15)
    except:
        # రైలు సంఖ్యకు తేదీలో రైలు లేకపోతే టైమ్‌ఔట్ జరుగుతుంది
        print("timeout for", train_number)
        return pd.DataFrame()
    if response.status_code == 200:
        page_content = response.text
        soup = BeautifulSoup(page_content, 'html.parser')
    else:
        print(train_number, 'Failed')
        return pd.DataFrame()
    table = soup.find("table")
    last_updated = None
    for thead in table.find_all("thead"):
        # thead పట్టికలో శీర్షిక వరుసను కలిగి ఉంటుంది
        last_row = thead.find("tr")
        # tr అనేది వరుస
        if last_row:
            td = last_row.find("td", colspan=True)
            if td and "Last Updated" in td.get_text(strip=True):
                last_updated = td.get_text(strip=True)
                break

    rows = []
    tbody = table.find("tbody")
    for tr in tbody.find_all("tr"):
        # td అనేది ప్రతి సెల్ (పట్టిక డేటా)
        tds = tr.find_all("td")
        if len(tds) != 4:
            continue
            # 4 వరుసలు ఉండాలి

        # --- స్టేషన్ సెల్ ---
        station_cell = tds[0]
        abbr = station_cell.find("abbr") # ఇది స్టేషన్ పేరు సంక్షిప్త రూపం
        station_name = abbr.get_text(strip=True) if abbr else station_cell.get_text(strip=True)
        station_code = abbr["title"] if (abbr and abbr.has_attr("title")) else None

        delay_small = station_cell.find("small") # ఇది ఆలస్య స్థితి
        delay_status = delay_small.get_text(strip=True) if delay_small else None

        # --- రాక సెల్ ---
        arrival_cell = tds[1]
        arrival_text = arrival_cell.get_text(" ", strip=True)  # ఉదాహరణకు, "02:11 PM / 02:11 PM" లేదా "Source"
        arrival_status_tag = arrival_cell.find("span")
        arrival_status = arrival_status_tag.get_text(strip=True) if arrival_status_tag else None

        # రాక సమయాలను పార్స్ చేయండి
        arr_sch, arr_act = None, None
        if "/" in arrival_text:
            parts = [p.strip() for p in arrival_text.split("/")]
            if len(parts) == 2:
                arr_sch, arr_act = parts

        # --- ప్రయాణ సెల్ ---
        departure_cell = tds[2]
        departure_text = departure_cell.get_text(" ", strip=True)  # ఉదాహరణకు, "02:12 PM / 02:12 PM" లేదా "Destination"
        departure_status_tag = departure_cell.find("span")
        departure_status = departure_status_tag.get_text(strip=True) if departure_status_tag else None

        dep_sch, dep_act = None, None
        if "/" in departure_text:
            parts = [p.strip() for p in departure_text.split("/")]
            if len(parts) == 2:
                dep_sch, dep_act = parts

        # --- PF సెల్ ---
        pf = tds[3].get_text(strip=True)

        rows.append({
            "Station Name": station_name,
            "Station Code": station_code,
            "Delay Status": delay_status,
            "Arrival Scheduled": arr_sch,
            "Arrival Actual": arr_act,
            "Arrival Status": arrival_status,
            "Departure Scheduled": dep_sch,
            "Departure Actual": dep_act,
            "Departure Status": departure_status,  # ఉదాహరణకు, "Destination"
            "PF": pf
        })

    df = pd.DataFrame(rows)
    df['train_number'] = train_number
    print(train_number)
    return df

    Google Maps API తో జియోకోడింగ్¶

    ఆలస్యాలను భౌగోళికంగా విజువలైజ్ చేయడానికి, ప్రతి స్టేషన్ కోసం అక్షాంశం మరియు రేఖాంశం అవసరం. స్టేషన్ పేర్లను సమన్వయాలలో మార్చడానికి Google Geocoding API ను ఉపయోగించాము:
    1. "స్టేషన్ పేరు (కోడ్) రైల్వే స్టేషన్, ఇండియా" వంటి క్వెరీస్‌ను నిర్మించడం, ఖచ్చితత్వాన్ని మెరుగుపరచడం.
    2. ఫలితాలు భారతదేశానికి సంబంధితంగా ఉండేలా ప్రాంతం బైయాస్ (region='in') ను వర్తింపజేయడం.
    3. JSON ప్రతిస్పందనలను పార్స్ చేసి lat మరియు lng విలువలను తీసుకోవడం.

    ఈ దశ ఆలస్యాలను భారతదేశం అంతటా మ్యాప్ చేయడానికి అనుమతించింది, ఆలస్యాల క్లస్టర్లwhere occur.

    import config
import requests

def get_lat_lng(location, gmaps_key=config.gmaps_key, region_bias = 'in'):
    """
    Returns (lat, lng) for a given location using Google Geocoding API.
    Raises informative exceptions on common failure modes.
    """
    url = "https://maps.googleapis.com/maps/api/geocode/json"
    params = {
        "address": location,
        "key": gmaps_key,
    }
    # Optional: region bias (e.g., "in" for India) to improve relevance
    if region_bias:
        params["region"] = region_bias

    response = requests.get(url, params=params, timeout=15)
    response.raise_for_status()  # network/HTTP-level errors

    payload = response.json()

    status = payload.get("status")
    if status != "OK":
        # Common statuses: ZERO_RESULTS, OVER_QUERY_LIMIT, REQUEST_DENIED, INVALID_REQUEST
        error_message = payload.get("error_message")
        return 0,0

    results = payload.get("results", [])
    if not results:
        return 0,0

    location_obj = results[0]["geometry"]["location"]  # guaranteed when status == OK
    lat = location_obj["lat"]
    lng = location_obj["lng"]
    return lat, lng

    DBSCAN తో క్లస్టరింగ్¶

    క్లస్టరింగ్ కోసం, మేము scikit-learn నుండి DBSCAN (Density-Based Spatial Clustering of Applications with Noise) ను ఉపయోగించాము. DBSCAN ఈ ఉపయోగం కేసుకు అనువైనది ఎందుకంటే:
    1. ఇది ముందుగా క్లస్టర్ల సంఖ్యను అవసరం లేకుండా సమాన ఆలస్య నమూనాలను కలిగిన స్టేషన్ల క్లస్టర్లను గుర్తిస్తుంది.
    2. శబ్దాన్ని సమర్థవంతంగా నిర్వహిస్తుంది, ప్రత్యేక ఆలస్య ప్రవర్తన కలిగిన అవుట్‌లయర్స్ (స్టేషన్లు) ను -1 అనే లేబుల్‌తో గుర్తిస్తుంది.
    3. ఆలస్య మేట్రిక్‌లతో కలిపి స్థల డేటాతో బాగా పనిచేస్తుంది.

    సరైన క్లస్టరింగ్‌ను నిర్ధారించడానికి, మేము DBSCAN ను వర్తింపజేయడానికి ముందు రాక ఆలస్యాలు మరియు స్టేషన్ సమన్వయాలను MinMaxScaler ఉపయోగించి సాధారణీకరించాము:
    1. eps = 0.07 (సాధారణీకరించిన యూనిట్లలో దూరం పథకం)
    2. min_samples = 5 (క్లస్టర్‌ను ఏర్పరచడానికి కనీస పాయింట్లు)

    ఫలితం: ఆలస్య తీవ్రత మరియు భౌగోళిక సమీపానికి ఆధారంగా స్టేషన్లు క్లస్టర్లలో సమూహీకరించబడ్డాయి, దీని ద్వారా దీర్ఘకాలిక ఆలస్యాల హాట్‌స్పాట్‌లు వెలుగులోకి వచ్చాయి.
